# LNA Website Sitemap

## Overview

This document defines every page on the LNA marketing website, its purpose, calls to action, and key content sections.

---

## / — Homepage

**Purpose:** Introduce LNA as an AI-powered document processing platform and drive visitors toward requesting a demo or exploring features.

| CTA       | Label          | Destination          |
| --------- | -------------- | -------------------- |
| Primary   | Request a Demo | /contact (demo form) |
| Secondary | See Features   | /features            |

**Key Sections:**

1. **Hero** — Headline, subheadline, primary + secondary CTAs, hero illustration/product screenshot
2. **Social Proof Bar** — Client logos or trust badges ("Trusted by 500+ teams")
3. **Problem Statement** — Pain points of manual PDF-to-Excel workflows
4. **Solution Overview** — How LNA automates extraction, mapping, and export
5. **Key Features Grid** — 3–4 highlight cards (Document Management, AI Extraction, Template Mapping, Data Q&A)
6. **Differentiator Section** — LLM-powered Q&A and Chat with Context Role spotlight
7. **How It Works** — 3-step visual flow (Upload → Extract → Export)
8. **Testimonials / Case Study Snippet** — Quote or metric from a customer
9. **Pricing Teaser** — Brief plan overview with CTA to /pricing
10. **Final CTA Banner** — "Ready to automate your document workflows?" + Request a Demo button
11. **Footer**
