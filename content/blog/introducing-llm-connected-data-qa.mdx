We're excited to announce one of the most requested features in LNA's history: **LLM-Connected Data Q&A** — a way to ask questions about your extracted document data using plain English.

## The Problem With Extracted Data

Extracting data from PDFs into Excel is a huge time-saver. But once you have structured data, the next question is always the same: _what does this data tell me?_

For most teams, answering that question means building pivot tables, writing formulas, or exporting to a BI tool. These steps take time, require specialized skills, and create friction between having data and understanding it.

## How Data Q&A Works

With LLM-Connected Data Q&A, you can simply ask:

- _"What's the total invoice amount for Q1?"_
- _"Which vendor had the highest shipping costs last month?"_
- _"Are there any duplicate entries in this batch?"_

LNA uses a large language model connected directly to your extracted data to generate accurate, contextual answers — with references back to the source documents.

## Built for Trust

Every answer from Data Q&A includes:

- **Source citations** — see exactly which documents the answer came from
- **Confidence indicators** — understand how reliable the answer is
- **Audit trail** — full traceability for compliance requirements

We designed this feature with the same philosophy that drives all of LNA: automation should be transparent and verifiable.

## Available on Professional and Enterprise Plans

LLM-Connected Data Q&A is available today on Professional and Enterprise plans. Existing Professional customers will see it in their dashboard automatically — no setup required.

If you're on the Starter plan and want to try it out, you can upgrade anytime from your account settings or [contact our team](/contact) for a demo.
